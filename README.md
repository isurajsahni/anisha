## Anisha Kumari — SEO & Digital Marketing Portfolio

Dark, modern, responsive portfolio built with Vite + React + TypeScript, TailwindCSS, and Framer Motion.

### Tech
- React 18 + TypeScript
- TailwindCSS
- Framer Motion
- Vite

### Design
- Background: Deep matte black (#0a0a0a)
- Accent: Vibrant Orange (#FF8C00)
- Typography: Inter (Google Fonts)

### Run locally
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
npm run preview
```
